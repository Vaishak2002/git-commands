> git init [to initialize]

>git status [to list]

>git add -A [to adding all untracked files to staging areas]

>git commit -m "commit message" [to commit added file tpo local repository]

>git clone [to take exact copy from remote repository to local workspace]

>git pull [to take changes from remote repository to local workspace]

>git push [to sent changes from local repository to remote repository]

>git config --list [to list configuration]

>git config --global user.name "username" [to add username]

>git config --global user.email "useremail" [to add usermail]
